# EDA Histogram App

An interactive Exploratory Data Analysis (EDA) web app built with Streamlit.

## Description
This app allows users to explore the distribution of numeric variables in the Tips dataset through an interactive histogram and summary statistics.

## Tools Used
- Python
- pandas
- matplotlib
- Streamlit

## How to Run
1. Install dependencies: `pip install streamlit pandas matplotlib`
2. Run the app: `streamlit run app.py`
